# Portal da Transparência Bolão 2026

Landing page estática do Portal da Transparência do Bolão da Copa 2026.

## Executar localmente

Abra `index.html` diretamente ou inicie um servidor local:

```bash
python3 -m http.server 8000
```

Depois acesse `http://localhost:8000`.

## Publicar no GitHub Pages

1. Crie um repositório no GitHub e envie estes arquivos para a branch `main`.
2. Em **Settings → Pages**, selecione **Deploy from a branch**.
3. Escolha a branch `main`, a pasta `/ (root)` e salve.

O site não possui dependências nem etapa de build.
